# DANDI Compute: AIND Ephys webpage

Simple webpage rendering the basic aspects of the **DANDI Compute: AIND Ephys** project.

## Development checks

- `npm run format:check` validates formatting.
- `npm run test:run` runs Vitest unit/integration tests.
- `npm run test:coverage` runs tests with coverage output (`coverage/lcov.info` for Codecov).
